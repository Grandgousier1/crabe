Delivery Note Transformer

Transformez vos bons de livraison scannés en PDF LaTeX harmonisés à l'aide de Gemini.

Configuration

Images du bon Ajoutez une ou plusieurs images. Laissez vide si vous fournissez un JSON structuré.

Hébergez ce dossier web/ sur Netlify en tant que site statique. Assurez-vous qu'un backend exécutant api_server.py est accessible via l'URL configurée et qu'il possède la variable GEMINI_API_KEY.
